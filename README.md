# Kubernetes Learning

Tài liệu học Kubernetes bằng tiếng Việt, được tổ chức theo lộ trình từ nền tảng đến vận hành production.

## Công nghệ

- Next.js 15
- Fumadocs
- Cloudflare Pages
- Mermaid

## Chạy local

```bash
npm install
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000). Trang chủ tự chuyển đến phần **Bắt đầu**.

## Build và deploy

```bash
npm run build
npm run preview
npm run deploy
```

## Cấu trúc nội dung

Tất cả tài liệu nằm trong `content/docs/`. Mỗi chủ đề có `meta.json` để khai báo đầy đủ thứ tự trang trên sidebar.

Hiện tại các trang là placeholder theo curriculum tổng thể. Có thể hoàn thiện lần lượt mà không cần thay đổi lại cấu trúc sidebar.
